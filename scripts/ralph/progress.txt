**Learnings for future iterations:**
- **Canvas over SVG for windgrams:** React 19 has peer dependency conflicts with visx. Canvas provides better performance for the dense grid visualization and avoids library compatibility issues.
- **Barometric formula for altitude:** pressureToAltitude() uses simplified standard atmosphere formula: h ≈ 44330 * (1 - (P/P0)^0.1903)
- **Theme integration pattern:** Extract HSL values from CSS custom properties (--foreground, --border) and convert to rgba for canvas rendering
- **Daylight hours filtering:** Filter atmospheric data to 6 AM - 8 PM local time for cleaner pilot-focused visualization
